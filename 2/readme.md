# 高端酒店粽子礼盒提货券H5页面需求文档

## 1. 项目概述

开发一个高端酒店粽子礼盒提货券H5页面，用于展示粽子礼盒信息和预约兑换券的登记。

## 2. 产品信息

### 2.1 主题与宣传语
- 主标题："浓情端午 棕享款"

### 2.2 产品详情
- 礼盒价格：88元/盒
- 礼盒内容：
  - 赤豆蜜枣粽 100克*1
  - 血糯蜜枣粽 100克*1
  - 蛋黄鲜肉粽 100克*1
  - 板栗鲜肉粽 100克*1
  - 咸鸭蛋 60克*4

### 2.3 提货券信息
- 提货券类型：实体券
- 编号范围：No.000000到No.000500
- 使用日期：2025年4月21日-5月31日
- 酒店地址：安徽省淮北市新型煤化工合成材料基地基地北路16号
- 订购热线：0561-3086666

## 3. UI设计要求

### 3.1 整体风格
- 设计风格：高端、典雅，符合酒店品牌形象
- 主色调：建议使用金色、红色等传统节日色彩，搭配酒店品牌色
- 字体：清晰易读，标题使用较大字号，正文保持舒适阅读体验

### 3.2 页面布局
- 顶部：酒店logo、活动主题banner
- 中部：
  - 礼盒图片展示（建议使用高质量实物图）
  - 礼盒内容详情（可使用图文结合方式展示各类粽子）
  - 价格信息（突出显示）
- 底部：
  - 提货券预约区域
  - 条款与细则
  - 酒店联系信息

### 3.3 响应式设计
- 确保页面在不同尺寸的移动设备上都能正常显示
- 针对主流手机屏幕尺寸（iPhone、Android等）进行适配
- 图片加载优化，确保快速加载

## 4. 功能需求

### 4.1 提货券预约功能
- 提交姓名、手机号码、购券数量，提交后系统返回券码
- 券码发放的有效范围内（No.000000到No.000500）
- 通过API提交信息到vika.cn后端

### 4.2 信息展示功能
- 礼盒详情展示，包含所有粽子类型和数量
- 可添加粽子详情介绍（可选，如原料、特色等）
- 提供酒店位置地图（可选，便于用户了解提货地点）

## 5. 交互流程

1. 用户访问H5页面
2. 浏览礼盒信息和活动详情
3. 输入信息预约获得券码
4. 系统返回验证结果
5. 显示提货相关信息（地点、时间等）

## 6. 技术实现

### 6.1 前端技术栈
- HTML5 + CSS3 + JavaScript
- 推荐使用Vue.js框架开发（轻量级且适合H5页面）
- 使用响应式设计框架（如Bootstrap或Vant UI）

### 6.2 后端集成
- 使用vika.cn提供的API接口获取数据
- API对接内容：
  - 提货券验证接口
  - 可选：提货记录更新接口

### 6.3 数据存储
- 在vika.cn创建数据表，包含以下字段：
  - 券码编号（唯一标识）
  - 姓名
  - 手机号码
  - 订购几套
  - 预约兑换时间
  - 备注信息（可选）
  - 登记时间（自动生成）

## 7. 条款与细则

页面底部需显示以下条款：
1. 此券不可兑换现金、不找零、不挂失、涂改撕毁无效；
2. 此券不能与其他优惠同时使用；
3. 请凭本券与指定日期内的提货时间到淮北相润金鹏酒店大堂吧领取粽子礼盒；
4. 此券需加盖本酒店印章，盖章生效，复印无效；
5. 此券最终解释权归淮北相润金鹏酒店所有。

## 8. 安全性考虑
- 数据传输加密：使用HTTPS协议
- 防止SQL注入：对用户输入进行过滤和验证

- 简易使用说明
